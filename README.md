# llm_rag_frontend

This project is a Vue 3 + Vite + Tailwind CSS + TypeScript front-end scaffold for the LLM RAG QA UI.

Quick Start

1. Install dependencies:

```bash
npm install
```

1. Run dev server:

```bash
npm run dev
```

1. Build for production:

```bash
npm run build
```

Notes

1. Tailwind v4 is configured with `tailwind.config.cjs` and `postcss.config.cjs`.
1. TypeScript support is enabled; run `npm run type-check` to validate types.
